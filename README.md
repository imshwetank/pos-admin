# POS Admin Template
HTML Admin Template For POS Apps

### Development
* clone repo / download repo
  ```
  git clone https://github.com/viandwi24/pos-admin
  ```
* download module
  ```
  npm install
  ```
  or
  ```
  yarn
  ```
* start dev server
  ```
  clear && yarn watch
  ```

### Library Use
* bootstrap
* tailwindcss
* PerfectScrollbar
* feather-icons
* chart.js
